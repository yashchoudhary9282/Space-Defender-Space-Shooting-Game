# Space-Defender
Space Defender is our first game project. It is a simple and fun 2D space shooter where the player controls a spaceship and shoots incoming asteroids to protect the ship from destruction.

Asteroids appear from different directions and move toward the player’s spaceship. The player must quickly aim and shoot them before they collide with the ship. Each destroyed asteroid increases the player’s score. As the game progresses, the difficulty increases with faster and more frequent asteroids.

This project helped us learn the basics of game development, including game loops, collision detection, player controls, and score tracking.
